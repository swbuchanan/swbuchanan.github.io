Voronoi Game

A Voronoi Diagram consists of a (possibly infinite) region of n-dimensional Euclidean space, and a set of distinguished points, each with an associated color. Every point of the region is then colored based on which of the distinguished points is closest (diagram of a 2-dimensional region from the Wikipedia article). In the game, two players start with an empty region of 2-space, and take turns placing one point at a time in the region. After a set number of turns, the game ends, and the player who has the most space in the region wins.
An obvious and interesting question to ask is what strategy, if any, can win the game. If the game is played in 1-space (there is an option for this on the linked site), then the second player has a winning strategy. Additionally, there is a one-round variation of this game where, instead of alternating placing one point a time for n rounds, the first player places n points, and then the second player places n points.
